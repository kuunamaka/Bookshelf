# Install
```sh
$ npm install 
```

# Run dev environment
```sh
$ npm run dev
```
access: http://localhost:3000/

# Compile code
```sh
$ npm run build
```

# Run unit-test
```sh
$ npm test
```

# Run formatter
```sh
$ npm run format
```
```sh
$ npm run lint
```

# Swagger UI
```sh
$ npm run swagger
```
access:  http://localhost:8080/
